Latest projects:

PkgPeeker

An application that leverages the possibilities offered by Web Containers to install npm packages in the browser and perform analysis on the node_modules folder to get useful and accurate information on the packages, their dependencies, file types, installation size, metadata, etc.

Built with React and Web Containers.

Code Playground

A CodePen clone. The application provides full-featured code editors to edit HTML, CSS and JavaScript, and view the result live in the browser using an iframe. It also provides a theme switcher, different layouts and support for external CSS stylesheets and JS libraries & scripts.

Built with React and CodeMirror.

Minesweeper

An attempt to build the famous Minesweeper game, playable in the browser, using only HTML, CSS and vanilla JavaScript. It offers the three standard difficulty levels as well as custom dimensions and mines count selection.
