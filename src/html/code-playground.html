ᐸCode Playgroundᐳ

Developed as a personal project, Code Playground is a web development tool that allows developers to write code (HTML, CSS & JavaScript) in the browser, and see the results in real time as they build.

As for the features, users can switch between a light and a dark theme, select between three different working layouts, maximize/minimize windows, add external CSS and/or JavaScript assets, while enjoying the modern, feature-packed embedded code editor.
