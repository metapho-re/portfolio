ᐸZahra's Portfolioᐳ

A personal portfolio tailored for a painter/artist, Zahra's Portfolio features a sleek, elegant design, smooth animated page transitions, easy navigation and user interactions.

In order to ease the inconvenience that stems from trying to view all the artworks and the related "click-to-open/click-to-close" hassle, an easy-to-use, "hoverable" thumbnail carousel was specifically designed, providing the best user experience while browsing through the art collections.
