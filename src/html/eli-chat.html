ᐸEli Chatᐳ

Developed as a personal project, Eli Chat is a live chat application featuring JSON Web Token-based authentication, real time features to instantly send/receive messages and see users getting online/offline, easy creation of chat groups with any number of signed up users, and of course data persistence.

The client app leverages React & React Native compatibility to share a common state management logic, while running on both web browsers and mobile devices.
